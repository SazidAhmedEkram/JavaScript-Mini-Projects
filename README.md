# JavaScript Mini Projects

A collection of beginner-friendly and interactive JavaScript mini projects built using HTML, CSS, and JavaScript.  
These projects are created for learning DOM manipulation, events, animations, and improving frontend development skills through practical implementation.

## Features
- Pure JavaScript projects
- Beginner-friendly code structure
- Interactive UI and animations
- Practice for DOM manipulation and event handling
- Easy to customize and expand

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Purpose
The main goal of this repository is to improve JavaScript skills by building small real-world projects and experimenting with different frontend concepts.

## Getting Started
1. Clone the repository
2. Open any project folder
3. Run the each .html file in your browser
